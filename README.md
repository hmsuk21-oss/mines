# mines# Minesweeper (지뢰찾기)

파이썬(Tkinter)로 만든 간단한 윈도우용 지뢰찾기 게임입니다.

## 실행 방법

1. `minesweeper.py` 파일을 다운로드하거나 복사하여 저장합니다.
2. 파이썬이 설치된 환경에서 아래 명령어로 실행하세요.

```bash
python minesweeper.py
```

## 게임 설명

- 마우스 **왼쪽 클릭**: 칸을 엽니다.
- 마우스 **오른쪽 클릭**: 깃발(플래그)을 설정/해제합니다.
- 모든 지뢰를 피해 빈 칸을 열면 승리합니다.
- 지뢰를 클릭하면 게임에서 집니다.

## 설정

- 기본 설정: 10x10 보드, 지뢰 10개
- 코드 상단에서 `rows`, `cols`, `mines` 값을 바꾸어 난이도를 조절할 수 있습니다.

## 필요 환경

- Python 3.x
- tkinter (일반적으로 Python 기본 탑재)
